# 🏙️ CivicLens — Community Hero

An AI-powered hyperlocal issue reporting platform built for **Vibe2Ship Hackathon** (Coding Ninjas × Google for Developers).

## 🌐 Live Demo
https://community-hero-2f711.web.app

## 📁 Problem Statement
Community Hero — Hyperlocal Problem Solver

## ✨ Features
- 📸 Photo-based issue reporting
- 🤖 Gemini AI Agentic Decision Chain (classify → severity → department routing → notification draft)
- 🔁 Autonomous duplicate detection & severity escalation
- 💬 Multi-turn Gemini Follow-up Q&A per issue
- ✅ AI Resolution Summary on status change
- 🧠 Gemini Issue Clustering for pattern detection
- 🗺️ Live Map with GPS-based issue pinning (Leaflet + OpenStreetMap)
- 📊 Impact Dashboard with status tracking
- 🏆 Reporter Leaderboard
- 📤 Shareable Report Cards
- 👍 Community Upvote system
- 📄 Export Report with AI executive summary

## 🚀 Setup
1. Go to https://aistudio.google.com/apikey and get a free Gemini API key
2. Open the app and enter your API key in the popup
3. Start reporting issues!

- Gemini 1.5 Flash API (via Google AI Studio)
